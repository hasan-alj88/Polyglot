# Rule 8.12 — Self-Chain Requires Numeric Indexing (Retired)
`PGE08012`

**Retired:** 2026-04-22. The `->` chain operator has been removed from Polyglot. Self-chains no longer exist. Use separate labeled `[-]` calls to invoke the same pipeline multiple times. See [[concepts/pipelines/chains]].
