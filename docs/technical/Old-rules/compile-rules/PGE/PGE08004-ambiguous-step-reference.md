# Rule 8.4 — Ambiguous Step Reference (Retired)
`PGE08004`

**Retired:** 2026-04-22. The `->` chain operator has been removed from Aljam3. Step references (`>N.param`, `>LeafName.param`) no longer exist. Labeled `[-]` calls with operation labels replace chains. See [[concepts/pipelines/chains]].
