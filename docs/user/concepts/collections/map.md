## #Map — Base Collection

`#Map` is the universal flat key-value collection, generated by the `{M} #Map` type macro. Two overloads exist: homogeneous `(<#, <#)` with both key and value types, and heterogeneous `(<#)` with key type only. Keys must be `##EnumLeafs` (predefined enum set); for arbitrary string keys, use `#Serial`.

See [[stdlib/types/collections]] for full macro definitions.

### Schema composition

`#Map` composes three `##` schema properties:

- `##Flat` — depth is 1 (one level of children)
- `##Homogeneous` or `##Heterogeneous` — depending on overload
- `##Sparse` — gaps allowed in keys, no ordering guarantee

### Access

Use `<` to access flexible children by key:

```polyglot
[r] $ages#map:string:int << ...
[r] $aliceAge#int << $ages<alice
[r] $bobAge#int << $ages<bob
```
