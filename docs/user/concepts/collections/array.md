## #Array ‚Äî Map Variant

`#Array` is generated by the `{M} #Array` type macro, which inherits from `#Map:#UnsignedInt:$ValueType` internally. It adds `##Contiguous` (no gaps, ordered) and `##Rectangular` (all branches at same depth have same child count). The optional `<Dim##Dimension` parameter defaults to `"1D"`.

See [[stdlib/types/collections]] for the full macro definition.

### Cartesian product keys

For a 1D array of length 3, the keys are `<0`, `<1`, `<2`.

For a 3x4 2D array, the key tree is a Cartesian product:

```
<0 ‚Üí <0, <1, <2, <3
<1 ‚Üí <0, <1, <2, <3
<2 ‚Üí <0, <1, <2, <3
```

`##Rectangular` ensures all branches at the same depth have the same child count.

### Access

```polyglot
[ ] 1D array access
[r] $first << $myArray<0

[ ] 2D matrix access ‚Äî branch 1, leaf 2
[r] $cell << $matrix<1<2

[ ] 3D cube access
[r] $voxel << $cube<2<3<0
```
