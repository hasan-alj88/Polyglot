# Training Lessons Learned

This directory contains records of specific edge cases, clarifications, and syntax nuances discovered during model training sessions (`pg_train_code` and `pg_train_lexer`). 

Whenever a piece of code or lexer token stream is corrected, the specific lesson learned should be recorded as a new markdown file in this directory and linked below. This helps the AI (and humans) maintain a reliable history of edge-case decisions.
