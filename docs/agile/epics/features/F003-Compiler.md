# Feature: Compiler Validation

## Objective
Traverse the structural tokens emitted by the lexer to build an AST and validate the syntactic and grammatical correctness of the Polyglot code before compilation.

## Requirements
- Consume the token stream generated by the Lexer.
- Construct the AST tree structure based on indentation markers.
- Validate semantic relationships (e.g. valid prefixes, valid marker combinations).
- Catch and report errors with precise line and column metadata.
