## New Topics Documented

### Enum Definitions Guide

**Core Concepts:**
- `{#}...{x}` enum definition block syntax
- `[A]` alias declaration for convenience references
- `[s]` serial load blocks (YAML, JSON, TOML)
- `[.]` field accessor for data mapping from loaded files
- Scope-wide `[s][!]` error handling (unique Polyglot feature)

**Key Features:**
- Multi-format support (YAML, JSON, TOML)
- Parallel file loading by default
- Nested field access with dot notation
- Configuration-driven enum patterns
- Complete working examples with real-world use cases

**Example Coverage:**
- Basic enum definitions
- Serial load from YAML/JSON/TOML
- Field accessor mapping
- Error handling patterns
- Multi-file loading
- Best practices and troubleshooting

### Trigger I/O Wiring Guide

**Core Concepts:**
- Trigger output parameters: `>output:type`
- Output-to-input wiring: `>trigger_out >> <pipeline_in`
- Pipeline input declaration: `[|] <input:type`
- Reactive data flow architecture

**Trigger Types Documented:**
- `|T.Folder.NewFiles` - File system monitoring
- `|T.CLI` - Command-line triggers
- `|T.HTTP.Request` - Webhook triggers
- `|T.Schedule.Cron` - Scheduled execution

**Example Coverage:**
- Basic trigger-to-pipeline wiring
- Multi-output triggers
- Complete log processor example
- Reactive patterns
- Best practices and troubleshooting
