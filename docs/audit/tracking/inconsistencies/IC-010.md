# IC-010: Missing "Wrappers" [W] in Vision Overview

**Severity:** Medium
**Tag:** definition
**File(s):** [[vision.md]], [[user/syntax/constructors.md]]
**Resolved Date:** 2026-04-23

## Description
`vision.md` lists the Polyglot Service pillars as "Trigger Monitor", "Queue Handler", and "Runner". However, Syntax files and `constructors.md` introduce a fourth pillar: **Wrappers [W]**. These are not defined or mentioned in the high-level Vision document.

## Resolution
**Rejected.** Wrappers are not a core architectural component of the Polyglot Service (unlike the Runner or Trigger Monitor). They are simply an execution construct within Polyglot Code (setup before the execution body, and cleanup after). `vision.md` correctly lists them under the "Polyglot Code" syntax elements, and they do not warrant promotion to the foundational architecture components list.
