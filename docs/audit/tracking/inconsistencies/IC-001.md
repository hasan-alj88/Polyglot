# IC-001: #VarState.Ready used in example

**Severity:** Critical
**Tag:** code
**File(s):** [[concepts/variable-lifecycle\|variable-lifecycle.md]]:85

## Description
`#VarState.Ready` was used in a code example but does not exist in the `#VarState` enum definition. Valid variants are: `.Declared`, `.Default`, `.Final`, `.Failed`, `.Released`.

## Resolution
Changed example to use `#VarState.Default`.
**Date:** 2026-03-30
