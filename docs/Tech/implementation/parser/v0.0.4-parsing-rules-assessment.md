## Test Failure Analysis

### Root Cause

**6 failing tests** in `polyglot-parser/src/validation.rs`:
1. `test_validate_serial_enum_with_default_handler_passes`
2. `test_validate_serial_enum_with_custom_handler_passes`
3. `test_validate_serial_enum_missing_handler_fails`
4. `test_validate_serial_enum_multiple_serial_blocks_with_handler_passes`
5. `test_validate_serial_enum_multiple_serial_blocks_missing_handler_fails`
6. `test_validate_serial_multiple_enumerations_mixed`

**Error Pattern:**
```
Parser: Unexpected token at X:1-1: expected definition ([#], [!], or [|]),
found indentation increase
```

### Problem Identification

1. **Syntax Version Mismatch:**
   - Tests use v0.0.3 syntax: `[#]`, `[<]`, `[~][s]`
   - Parser expects v0.0.4 syntax: `{#}`, `[.]`, indented field mappings

2. **Indentation Expectation Mismatch:**
   - Tests expect flat structure (v0.0.3)
   - v0.0.4 requires indented field mappings under `[s]`
   - Parser rejects indentation in enum context (not implemented)

### Example

**Test Expects (v0.0.3):**
```polyglot
[#] #Config
[<] .timeout: pg.int << 30
[~][s] <~ .timeout: pg.int
[s] |YAML.Load"config.yaml"
[X]
```

**Should Be (v0.0.4):**
```polyglot
{#} #Config;Default
[A] #Config

[s] |YAML.Load"\\FileDir\\config.yaml"
   [.] .timeout:pg.int << .timeout    # ← Indented under [s]

[s][!] !*
{x}
```

```rust
fn parse_serial_load_in_enum(&mut self) -> Result<SerialLoad, ParserError> {
    // [s] |Format.Load"path"
    self.expect(TokenKind::MarkerSerialLoad)?;

    let format = self.parse_serial_format()?;
    let file_path = self.parse_expression()?;

    // Expect indentation for field mappings
    if self.check(TokenKind::Indent) {
        self.advance();

        let mut field_mappings = Vec::new();
        while !self.check(TokenKind::Dedent) && !self.at_end_of_block() {
            field_mappings.push(self.parse_field_mapping()?);
        }

        if self.check(TokenKind::Dedent) {
            self.advance();
        }
    }

    // Continue parsing...
}

fn parse_field_mapping(&mut self) -> Result<FieldMapping, ParserError> {
    // [.] .field:type << .source.path
    self.expect(TokenKind::MarkerSubfield)?;

    let field_name = self.expect_field_name()?;
    let field_type = self.parse_type()?;

    self.expect(TokenKind::OpPullFinal)?;  // <<

    let source_path = self.parse_field_accessor()?;

    // Check for nested subfields (additional indentation)
    let mut subfields = Vec::new();
    if self.check(TokenKind::Indent) {
        self.advance();

        while !self.check(TokenKind::Dedent) {
            subfields.push(self.parse_field_mapping()?);  // Recursive
        }

        if self.check(TokenKind::Dedent) {
            self.advance();
        }
    }

    Ok(FieldMapping {
        field_name,
        field_type,
        source_path,
        subfields,
        span: self.span(),
    })
}
```
