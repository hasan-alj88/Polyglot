# Polyglot — Brand & Logo Creative Brief

*Created: 2026-04-14*

---

## About the Project

**Polyglot** is the first parallel programming automation language — trigger-driven, cross-language, and safe by construction. It lets developers orchestrate workflows across Python, Rust, JavaScript, C++, and other languages from a single unified syntax where every pipeline is event-triggered with compiler-enforced safety.

**Brand position:** "The serious one." Polyglot brands on conviction, not hype. The design philosophy is that "happy path only" code is irresponsible — every error, race condition, and permission must be accounted for before anything runs.

**Brand sentence:** The trigger-driven language that makes your existing code work together — and proves it at compile time.

**Author origin:** Bahrain

---

## Three Messaging Pillars

1. **"Your code already works. Let it."** — Polyglot doesn't replace Python, Rust, or JS. It honours existing ecosystems and makes them work together.
2. **"Compile-time, not 3am."** — If it compiles, entire classes of bugs don't exist.
3. **"There is no main()."** — Every pipeline reacts to events. There is no sequential entry point.

## Logo Mark: `{-}`

The primary mark is the characters `{-}` — the pipeline definition syntax in Polyglot Code. Pipelines are why Polyglot exists. It's what developers type most and see first.

### Design direction
- Render `{-}` as a **unified glyph**, not three separate characters
- Monospace or geometric sans-serif base
- The curly braces `{ }` should feel **structural**, not decorative — they are syntax
- The dash `-` connects them — a pipeline bridging two boundaries
- Must work at **16px** (favicon) through billboard size
- Clean, minimal, geometric — no ornamentation

### Why `{-}`
- `{-}` defines a pipeline — the core unit of Polyglot
- It's typeable, memeable, works as favicon, GitHub avatar, sticker
- Developers will call it "the curly dash" — that's free brand recognition
- Precedent: Rust has `fn`, Go has the gopher. Simple marks win.

## Colour Palette

### Primary: Deep Teal Green
- **Hex:** `#0D7377` (adjust to taste)
- **Meaning:** The Tree of Life (Shajarat al-Hayat) — a 400-year-old tree standing alone in Bahrain's desert with no visible water source. Represents resilience, growth in hostile conditions, finding resources where others can't. Polyglot thrives where cross-language orchestration is hostile.

- **Meaning:** Bahrain's pearl diving heritage — purity, clarity, craftsmanship. Pearls are formed through sustained effort, not speed. Polyglot values correctness over velocity.

### Dark: Charcoal
- **Hex:** `#1A1A2E`
- **Meaning:** "The serious one." Confidence, depth, authority.
- **Usage:** Dark mode backgrounds, body text, code blocks

- **Hex:** `#C4A35A`
- **Meaning:** Bahraini desert sand — warmth, heritage, endurance.
- **Usage:** Highlights, accent elements, hover states, special callouts

### Colour Ratios (approximate)
- 60% Pearl White / Charcoal (depending on mode)

## Typography Direction

- **Headings:** Geometric sans-serif (think: Inter, IBM Plex Sans, or similar)
- **Code:** Monospace that renders `{-}` cleanly (JetBrains Mono, Fira Code, or similar)
- **Body:** Clean sans-serif matching headings
- **Characteristics:** No rounded/bubbly type. This is a serious tool. Legible, precise, structured.

## Extended Brand System: The `%` Tree

Beyond the `{-}` logo mark, the **`%` metadata tree** serves as the visual system for documentation, presentations, and the website. In Polyglot, everything is a tree rooted at `%`:

```
%
 |- %#   structs (data types)
 |- %-   pipelines (execution)
 |- %T   triggers (events)
 |- %=   expanders (parallelism)
 |- %*   collectors (reconciliation)
 |- %$   variables (state)
 |- %W   wrappers (environments)
 |- %Q   queues (dispatch)
 |- %!   errors (safety)
 |- %@   packages (imports)
 |- %_   permissions (security)
```

### Block type icons
Each `{X}` definition block gets its own icon in the brand system, all derived from the teal palette:

| Block | Symbol | Represents | Suggested icon element |
|-------|--------|-----------|----------------------|
| `{@}` | Package | Foundation | Root/anchor |
| `{#}` | Struct | Data | Grid/structure |
| `{-}` | Pipeline | Execution | Flow/arrow (this IS the logo) |
| `{T}` | Trigger | Events | Lightning/signal |
| `{W}` | Wrapper | Environment | Shield/enclosure |
| `{Q}` | Queue | Dispatch | Stack/lane |
| `{!}` | Error | Safety | Alert/guard |
| `{_}` | Permission | Security | Lock/key |
| `{*}` | Collector | Reconciliation | Funnel/merge |
| `{=}` | Expander | Parallelism | Fork/split |
| `{;}` | Environment | Runtime | Gear/container |
| `{N}` | Native | Host code | Chip/core |

## Mascot (Optional, Secondary)

**Concept:** An anthropomorphized Tree of Life with Polyglot syntax integrated into its form.

- The trunk echoes the `{ }` curly brace shape
- Branches have `[ ]` square bracket angles (control flow)
- Leaves/canopy are `( )` rounded shapes (IO, data)
- Friendly but **not cartoonish** — fits "the serious one"
- Teal green colour with desert gold accents
- Stands on sandy/desert ground (Bahraini landscape)

**Personality:** Quiet confidence. The tree doesn't shout — it just keeps growing where nothing else can. It's been here for centuries and it'll be here for centuries more.

**Usage:** Documentation illustrations, community merch, conference materials, error pages ("this tree has seen worse"), onboarding guides.

---

## Brand Don'ts

- No gradients on the `{-}` mark itself — keep it flat and structural
- No rounded/bubbly typography
- Don't make the braces decorative — they are literal syntax
- No neon, no web3/crypto aesthetics
- Don't compare to workflow engines (Airflow, Temporal) — Polyglot is a language, not an engine
- Don't lead marketing with "parallel programming" — lead with outcomes (safety, reuse)
- No "low-code" or "no-code" positioning — Polyglot is proudly code-first
- Don't soften the opinionatedness — exhaustive coverage is the point

## Reference: Bahraini Heritage Elements

**Tree of Life (Shajarat al-Hayat)**
- 400+ year old Prosopis cineraria tree
- Stands alone in the desert south of Bahrain, no visible water source
- Symbol of resilience and life in harsh conditions
- Wide canopy, thick trunk, growing from bare sandy ground

**Pearl Diving (Ghaws)**
- Bahrain was the historic centre of the pearl trade
- Pearls represent: patience, craftsmanship, hidden value beneath the surface
- The pearl diver goes deep to find something precious — like Polyglot reaching into language runtimes to find reusable code

These elements inform the colour palette and brand story, not the logo geometry.
