<objective>
## Goal
Fix operation label marker from `($) $Label` to `(-) $Label` under `[-]` pipeline calls across all documentation. The IO parameter handling marker must mirror the prefix of its parent context: `-` pipeline → `(-)`.

## Purpose
The IO mirroring rule is a core Polyglot design principle. Operation labels currently violate it by using `($)` under pipeline calls instead of `(-)`. This creates inconsistency with the grouped fallback syntax (Plan 01) which correctly uses `(-) $label`.

## Output
- All `($) $Label` declarations under `[-]` pipeline calls changed to `(-) $Label`
- EBNF grammar updated
- Glossary and blocks.md updated
- Compile rule examples corrected
</objective>
