## Generation approach

PGE + PGW indexes were generated by a bash script that:
1. Walked `*.md` files in each directory, extracting `code:` and `name:` from frontmatter.
2. Grouped by the first two digits of the code (thousand-range).
3. Emitted a range heading + a three-column table (`Code | Name | Link`) per range.
4. Appended a "Range Categories" summary table at the foot.

The three narrative indexes (root compile-rules, algorithms, compiler) were hand-written because each has unique prose and a small enough file list to curate.
