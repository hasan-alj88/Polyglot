<task type="auto">
  <name>Task 1: Add §12.3 Reassemble Operators to EBNF grammar</name>
  <files>docs/technical/ebnf/12-collections.md</files>
  <action>
    After §12.2 Collect Operators, add new section:

    ### 12.3 Reassemble Operators (`=*`)

    Add EBNF productions:
    - reassemble_line: execution marker + reassemble_invocation + NEWLINE + expander IO + collector IO
    - reassemble_invocation: '=' '*' reassemble_operator
    - reassemble_operator: reassemble_agg | reassemble_into
    - reassemble_agg: "Agg.Sum" | "Agg.Count" | "Agg.Average" | "Agg.Max" | "Agg.Min" | "Agg.Concatenate"
    - reassemble_into: "Into.Array" | "Into.Map" | "Into.Dataframe"
    - reassemble_io: "(=)" for expander input + "(*)" for collector output

    Add rules:
    - =* has no intermediate body — expander feeds directly to collector
    - Compiler expands =* to equivalent = + * pair
    - IO brackets: (=) for expander input, (*) for collector output

    Add Reassemble IO Signatures table with all 9 operators, their inputs, outputs, and equivalent expand-collect pair.
  </action>
