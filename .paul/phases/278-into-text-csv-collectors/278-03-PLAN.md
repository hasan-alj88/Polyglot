    - Frontmatter: audience pg-coder, type specification, status draft, updated 2026-04-11
    - Description: k-way merge on CSV, row-aware with header preservation
    - Syntax: [-] *Into.CSV.Merge with (*) <diffs #TextDiffs, (*) <base #string, (*) <conflict #MergeStrategy (optional, default .FirstWins), (*) >result #MergeResult

    - Permissions: _File.TempWrite (only when overflow spills to disk)
    - Related: link to Into/INDEX, Text.Diff, Into/Text.Merge, MergeStrategy, MergeResult, ForEach/CSV.Rows
