<objective>
## Goal
Document the 4 text operators: =Text.Diff comparator pipeline, =ForEach.Text.Lines expander, *Into.Text.Append collector, and *Into.Text.Merge collector.

## Purpose
These operators complete the expand/collect pattern for text data — users can split text into lines, process them in parallel, and reassemble via append (simple concatenation) or merge (k-way diff-based merge with conflict resolution). The =Text.Diff comparator enables the merge workflow by producing #TextDiffs.

## Output
- 4 new operator documentation files
- 2 updated INDEX files (ForEach/INDEX.md, Into/INDEX.md)
</objective>
