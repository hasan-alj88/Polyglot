<objective>
## Goal
Audit all 1,908 documentation files, remove outdated/duplicate content, and consolidate into a clean, navigable structure with a single authoritative index.

## Purpose
The docs/ directory has accumulated redundant content across multiple reorganizations — 778 files in archive/, 463 in _backups/, duplicate User/ directories nested inside other directories, and multiple competing index files. A clean documentation base is prerequisite for writing the language specification (Phase 2).

## Output
- Clean docs/ structure with ~200-400 files (down from 1,908)
- Single authoritative INDEX.md
- Documentation standards document
</objective>
