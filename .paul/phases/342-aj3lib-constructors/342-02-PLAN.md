## Goal
Define Tier 2a `{$}` constructors for `$Re`, `$MIME`, and `$Dur` — three constructors with simpler type structures. Create corresponding `{#}` type definitions for `#Re` and `#MIME` (the `#Duration` type already exists). Create `-Re.Parse`, `-MIME.Parse`, and `-Dur.Parse` runtime pipelines.

Continues issue #342 Tier 2 constructor catalog. Tier 2b ($Ver, $URL, $IP, $Color) will follow in plan 342-03 with more complex multi-field struct types.
