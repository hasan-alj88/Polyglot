<objective>
## Goal
Define Tier 1 `{$}` constructor blocks for `$DT` and `$Path` — the two constructors that replace existing inline pipeline notation (`-DT"..."` and `-Path"..."`). Create corresponding `-DT.Parse` and `-Path.Parse` runtime pipelines. Establish the `docs/user/aj3lib/constructors/` directory.

## Purpose
Issue #342 Tier 1 replaces inline pipelines in execution body with compile-time-guaranteed constructors (per #341 design). `$DT` and `$Path` are the highest-priority constructors because they replace existing syntax that users already write. This plan creates the aj3lib constructor catalog foundation.

## Output
- Constructor directory with INDEX.md registry
- `$DT` constructor with ~7 overloads (ISO-8601 full, date-only, time-only, keywords)
- `$Path` constructor with ~2 overloads (absolute/relative paths)
- `-DT.Parse` and `-Path.Parse` runtime pipeline docs
- Updated DT and Path pipeline INDEX/docs to reference constructors
</objective>
