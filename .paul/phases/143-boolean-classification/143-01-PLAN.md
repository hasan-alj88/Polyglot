<objective>
## Goal
Introduce `##Enum` as a schema classification for struct types whose fields are all enum fields (no `#type` annotation). Replace the vague "enum struct" prose with the formal `##Enum` schema. Apply to `#Boolean` and all enum types.

## Purpose
The term "enum struct" is imprecise — all `{#}` types are structs, so "enum struct" adds confusion. `##Enum` as a schema (like `##Scalar`, `##Flat`) gives enums a formal classification within the existing schema system. This resolves the inconsistency where `#Boolean` is called "enum struct" in some places and just "enum" in others.

## Output
- `##Enum` schema definition added to approved schemas
- All "enum struct" references replaced with `##Enum` classification
- `#Boolean` and pglib enums consistently classified via `##Enum`
</objective>
