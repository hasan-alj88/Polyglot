## Goal
Create PGE14xxx compile error code specifications for `{$}` Constructor block validation — 7 definition-time errors (PGE14001-14007) and 4 call-site errors (PGE14010-14013).

## Purpose
Completes the compiler rule infrastructure for the {$} constructor system introduced in #341/#342. Without these error codes, the constructor contract (guaranteed Final, no error surface) has no formal enforcement specification.

## Output
- 11 new PGE14xxx error specification files in `docs/technical/compile-rules/PGE/`
- Updated COMPILE-RULES.md index with `### 14 — Constructors` category
</objective>
