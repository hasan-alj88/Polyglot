<objective>
## Goal
Add `#bytes` and `#dt` binding types to all three marshalling/encoding tables and update the `#NativeType` enum definition to include byte-equivalent fields for all four languages.

## Purpose
Two types exist in the wire protocol (native-dispatch.md) but are missing from the SDK binding tables. `#bytes` has no Polyglot type row at all (listed as `(none)`). `#dt` exists in native-dispatch.md but uses ISO 8601 format — the brainstorm decided on epoch seconds for universality. Both are absent from polyglot-sdk.md. Closing this gap ensures all three documents agree on the full set of binding-compatible types.

## Output
Three updated documentation files with consistent `#bytes` and `#dt` rows across all tables.
</objective>
