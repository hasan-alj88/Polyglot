<objective>
## Goal
Create the implementer-facing job sandbox specification, expand the user-facing enforcement docs, and add the Permission Manifest structure to the Behavior Contract.

## Purpose
Issue #313 specifies how the Polyglot Runner translates compiled {_} permissions into OS-level restrictions (Landlock, seccomp-bpf, Linux namespaces, cgroups v2) applied per-job before foreign code executes. This plan documents the core sandbox architecture — the foundation that compile rules (plan 313-02) and future sub-issues (#314 resource categories, #315 OTel) build on.
