<objective>
## Goal
Add the complete `=RT` runtime execution subsystem to the Polyglot aj3lib specification: `=W.RT` wrapper (#76), `!RT` error namespace (#77), and `=RT.*` execution pipelines (#78).

## Purpose
Polyglot's cross-language integration pillar requires the ability to execute foreign code. This plan delivers the full stack: runtime wrappers that manage language environments, error types for runtime failures, and seven execution pipelines covering all code execution patterns (Function, Script, CLI, Bind × Inline/File).

- 1 new file: `docs/user/aj3lib/pipelines/RT.md`
- 6 edited files: W.md, types.md, errors.md, INDEX.md, AJ3LIB.md, blocks.md
