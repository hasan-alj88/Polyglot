Issues #94, #93, #79 converge on this redesign. Ground truths + macros are the foundation that `=#.*` validation pipelines (Plan 94-02/04) build on. Resolves #93 (generic Dataframe) directly.

## Output
9 spec files updated with macro-based type system. No new files created in this plan.
</objective>
