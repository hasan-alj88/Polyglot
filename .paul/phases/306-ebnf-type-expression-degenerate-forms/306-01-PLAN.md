<objective>
## Goal
Resolve 4 EBNF edge cases (X.37-X.40) for type expression degenerate forms by tightening grammar productions and cleaning up stale `#*` / `[:] :*#*` references.

## Purpose
These grammar holes allow meaningless or ambiguous type annotations to parse. Tightening the EBNF at the grammar level prevents degenerate forms before the compiler runs, reducing the need for semantic error rules.

## Output
- Updated EBNF §4.1 and §3.3 with tightened productions
- New edge cases EC-4.20 through EC-4.23
- PGE04025 reduced to stub (grammar now enforces mandatory element type)
- Stale `#*` and `[:] :*#*` references cleaned from ~5 files
</objective>
