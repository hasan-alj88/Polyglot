<objective>
## Goal
Add `~ForEach.Map` expand operator and `*Into.Map` collect operator to the Polyglot language specification. Restructure ForEach and Into stdlib docs from single files into folder hierarchies where each variant gets its own file (dot = folder).

## Purpose
`#Map` is a documented collection type (unordered, sparse, flat key-value) but has no dedicated expand/collect operators. Adding Map-specific operators completes the collection operator matrix. The file restructure establishes the convention that namespace dots map to folder hierarchy.

## Output
- New folder structures: `ForEach/` and `Into/` with per-variant files
- Original `ForEach.md` and `Into.md` deleted
- Updated: collections.md, INDEX.md, EBNF.md, PGE03007, PGE03008
</objective>
