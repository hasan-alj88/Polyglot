## Design Decisions (from brainstorm)
- String serialization — same as SDK/native-dispatch, not FFI/direct memory
- Dual [W] wrapper — setup top-to-bottom, cleanup bottom-to-top, skip if running
- ;Caller;Callee on [-] line — first env = caller, second = callee
- <arg#array.Variable — positional; <kwarg#Record.String.Variable — named
- Errors extend !Run.* namespace
- One direction per pipeline — trigger source = caller
- ##Enum propagates %##Active.One — one language branch per Variable instance
- -Variable.Convert is {N} native but user-callable
- Extensible pair framework — language pairs added as Rust native ops
</context>
