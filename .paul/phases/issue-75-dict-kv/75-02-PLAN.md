    Rewrite types.md using docs/draft.md as the authoritative source. Key changes:

    1. **Type Annotation section** — Replace ; with # throughout:
       - `$IntValue#int <~ 42` (not ;int)
       - Character role table: # = Annotation (what type it holds), remove ; row
       - Document rule: "# starts type context; nested type refs drop # prefix"
       - Examples: `$users#array:Person`, `$map#dict:string:int`

    2. **Basic Types section** — Restructure as layered hierarchy:
       - Layer 0: RawString (compiler intrinsic, no {#} definition)
       - Layer 1: #String with .string#RawString and .re#RawString <~ ".*"
       - Layer 2: All scalar subtypes (#Int, #UnsignedInt, #Float, #Sci, #Eng, #Dimension)
         - Each with [#] <~ #String inheritance
         - Each with %Alias and final .re regex
         - Include #Sci and #Eng (no longer TBD/deferred)
       - Layer 2b: #Boolean (NOT a #String subtype — independent enum struct)

    3. **New section: Schema Properties** — Document [#] % syntax:
       - %Key.Type, %Key.Gap, %Ordered, %Depth.Max, %Alias
       - %Open is REMOVED — do not include
       - %Depth.Max inference model (counts : flexible levels only)

    4. **New section: Generic Type Parameters** — Document < syntax:
       - `{#} #Array<ValueType<Dim` pattern
       - Positional binding via : separators
       - [<] constraint blocks under type params
       - `#*` wildcard type

    5. **Element-Typed Arrays** — Change . to : notation:
       - `$files#array:path` not `$files;array.path`
       - `$matrix#array:float:2D` not `$matrix;array.float:2D`
       - Access: `$items:0` not `$items.0`

    6. **Struct Types** — Update type annotation rule:
       - "In type annotations (after #), nested type refs drop the # prefix"
       - `$user#UserRecord` not `$user;UserRecord`

    7. **All remaining sections** — Replace ; with # in type positions:
       - Typed Flexible Fields: `[:] :*#Type` not `[:] :*;Type`
       - Type Conversions: `$data#serial` not `$data;serial`
       - Live Type Modifier: `#live.#PipelineStatus` not `;live.#PipelineStatus`
       - Path Type: `$AppDir#path` not `$AppDir;path`
       - Struct examples: `.timeout#int` not `.timeout;int`

    Preserve: Obsidian frontmatter, wikilinks, @-imports, Mermaid diagrams (if any).
    Preserve: All existing sections not listed above (String Interpolation, Path details, etc.)
    Do NOT change any compile rule references (PGE-xxx codes stay as-is).
    Follow docs/audit/ rules for documentation style.
