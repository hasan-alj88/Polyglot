<objective>
## Goal
Fix array access syntax in arrays.md (`:` → `<`) and add a brief note linking `:ND` declaration to `<` access.

## Purpose
arrays.md uses `:` for element access ($matrix:0:1) which contradicts the tree child accessor `<` used everywhere else ($matrix<0<1). The `:ND` declaration vs `<` access relationship is undocumented.
</objective>

</context>
