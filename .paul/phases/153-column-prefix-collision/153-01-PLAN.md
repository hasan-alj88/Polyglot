## Goal
Document the `=#.*` stdlib namespace pattern in identifier docs and clarify it in the stdlib reference, resolving the undocumented prefix collision flagged by issue #153.

## Purpose
The `=#.*` namespace (10 pipelines for schema operations) follows the same pattern as `=W.*`, `=Q.*`, `=T.*` but is never explained in identifier rules. Users may misread `=#.Column` as a prefix collision.

## Output
- Updated `docs/user/syntax/identifiers.md` with stdlib namespace documentation
- Updated `docs/user/stdlib/pipelines/#.md` with namespace clarification
</objective>

    - `=W.*` — Wrappers
    - `=Q.*` — Queue operations
    - `=T.*` — Triggers
