## Goal
Update types.md and data-is-trees.md to reflect the new three-tier prefix system (#/##/###), schema types, field types, `<` operator, type parameter defaults, and #IndexString.

## Purpose
Issue #88 introduces a declarative schema system for Polyglot's tree-based data types. The type system spec must document the three-tier prefix system, all approved ## schema types, ### field types, the `<` accessor/parameter operator, and the complete type hierarchy from Layer 0 (RawString) through Layer 2 (scalars + #IndexString + #Boolean).

- Updated `docs/user/concepts/data-is-trees.md` with leaf-only invariant and schema concepts
