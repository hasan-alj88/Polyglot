## Key Decisions Made This Session

1. Three `{Q}` block types: `#Queue` (config), `#QueueRules` (queue policy), `#JobRules` (job policy)
2. Scope rule: read anything, act only on your scope
3. `[?]` IF, `[&]` AND, `[+]` OR — standard Polyglot conditionals, not queue-specific
4. State guards mandatory (compiler error without them)
5. `(#)` parameters with `<~` defaults, capturable as `$` variable objects
6. Free.RAM split: `.Soft` (memory.high hint) vs `.Hard` (memory.max guarantee)
7. Pause reason set: resume only when ALL rules agree (no rule wants pause)
8. `=*` reassemble operator: general-purpose (not queue-only), tracked in #283
9. `-Q.DoNothing`: real pglib pipeline, compiler warns on uncovered states
10. Snapshot = fork capability with compiler safety rules
11. Old files get `@d:` deprecation markers (not deleted)
12. Cross-queue constraints moved from pipeline IO to `#Queue` properties
13. `Dispatch.Wait.TimeOut` redesigned into `#QueueRules` model
